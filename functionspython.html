Python Functions & OOP Cheat Sheet

Quick reference for functions, lambda, classes, inheritance

Functions

  • Define: def greet(name): return f"Hello {name}"
  • Call: greet("Alice")
  • Default arguments: def add(a, b=5)
  • Variable arguments: def sum(*args)
  • Keyword arguments: def person(**kwargs)

Lambda Functions

  • Syntax: lambda x, y: x + y
  • Example: add = lambda a,b: a+b
  • Used with map, filter, reduce

Classes

  • Define: class Person:
  • Constructor: def __init__(self, name): self.name=name
  • Methods: def greet(self): print(f"Hello {self.name}")
  • Create object: p = Person("Alice")
  • Call method: p.greet()

Inheritance

  • Single inheritance:
    • class Student(Person):
    def __init__(self, name, roll):
        super().__init__(name)
        self.roll = roll
  • Multiple inheritance:
    • class C(A, B): pass
