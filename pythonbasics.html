Python Basics Cheat Sheet

Quick reference for Python fundamentals

Variables & Data Types

  • Declare variable: x = 10
  • Data types: int, float, str, bool, list, tuple, dict, set
  • Check type: type(x)
  • Multiple assignment: a, b = 5, 10

Input / Output

  • Print: print("Hello World")
  • Input: name = input("Enter your name: ")
  • Formatted print: print(f"Hello {name}")

Operators

  • Arithmetic: + - * / % ** //
  • Comparison: == != > < >= <=
  • Logical: and or not
  • Assignment: = += -= *= /= %=
  • Membership: in, not in
  • Identity: is, is not

Type Conversion

  • Integer: int("10") → 10
  • Float: float("10.5") → 10.5
  • String: str(10) → "10"
  • Boolean: bool(0) → False
  • List / Tuple / Set conversion: list((1,2)) → [1,2]
